1. Variables: Almacenando Información

Una variable es un espacio con nombre en la memoria del ordenador donde guardas un dato. Piénsalo como una caja etiquetada: pones algo dentro (el valor) y le pones una etiqueta (el nombre de la variable) para encontrarlo fácilmente. Para crear una, usas el signo igual =.

nombre_de_la_variable = valor

# Ejemplo:
puntuacion_jugador = 100

Al nombrar variables, hay reglas: deben empezar con una letra o guion bajo, no pueden tener espacios ni caracteres especiales (salvo _), y distinguen mayúsculas de minúsculas. Por convención, en Python se usa snake_case (palabras en minúscula separadas por guiones bajos).

2. Tipos de Datos Básicos

Un tipo de dato le dice a Python qué clase de información es una variable y qué se puede hacer con ella. Los cuatro tipos fundamentales son:

  • Cadena (str): Para texto. Se escribe entre comillas ' ' o " ". Ejemplo: "Hola".
  • Entero (int): Para números enteros sin decimales. Ejemplo: 42, -100.
  • Flotante (float): Para números con decimales. Ejemplo: 3.14, 19.99.
  • Booleano (bool): Representa verdad o falsedad. Solo puede ser True o False (con mayúscula inicial).

Puedes usar la función type() para saber de qué tipo es una variable.

3. La Función print()

La función print() es tu herramienta para mostrar mensajes o resultados en la pantalla. Para combinar texto y variables, la forma más moderna y recomendada son las cadenas-f. Simplemente pon una f antes de la comilla de apertura y encierra tus variables entre llaves {}.

nombre = "Carlos"
puntuacion = 85
# Usando una cadena-f
print(f"El jugador {nombre} tiene {puntuacion} puntos.")

4. La Función input()

Para hacer tus programas interactivos, usas la función input(). Detiene el programa, muestra un mensaje y espera a que el usuario escriba algo y presione Enter.

La Regla de Oro: ¡input() siempre devuelve una cadena de texto (str)! No importa si el usuario escribe un número, para Python inicialmente será texto. Esto es crucial y nos lleva al siguiente punto.

nombre = input("¿Cómo te llamas? ")
print(f"Hola, {nombre}!")

5. Conversión de Tipos (Casting)

Como input() devuelve texto, no puedes hacer operaciones matemáticas directamente. Primero debes convertir ese texto a un tipo numérico. Este proceso se llama casting.

  • int(valor): Convierte el valor a un número entero.
  • float(valor): Convierte el valor a un número con decimales.
edad_texto = input("Introduce tu edad: ") # Recibimos "25"
edad_numero = int(edad_texto) # Convertimos "25" a 25
edad_futura = edad_numero + 10 # ¡Ahora sí podemos sumar!
print(f"En 10 años tendrás {edad_futura} años.")
