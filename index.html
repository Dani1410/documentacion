Guía de Entorno para tu Proyecto

Una guía interactiva para configurar paso a paso el entorno de desarrollo de tu simulador de inversiones.

Comenzar Configuración

Paso 1: Instalar Python

Python es el lenguaje principal para tu proyecto. Asegúrate de tener la última versión estable. Selecciona tu sistema operativo para ver las instrucciones específicas.

Instalación en Windows

  1. Ve a python.org/downloads y descarga el instalador.
  2. Ejecuta el instalador.
  3. ¡Muy importante! Marca la casilla "Add python.exe to PATH".
  4. Selecciona "Install Now" y completa la instalación.
  5. Verifica la instalación abriendo CMD o PowerShell y ejecutando:
python --version

Instalación en macOS (con Homebrew)

  1. Abre la Terminal e instala Homebrew si no lo tienes.
  2. Instala Python con el siguiente comando:
brew install python

Verifica la instalación ejecutando:

python3 --version

Instalación en Linux (Ubuntu/Debian)

Abre una terminal y ejecuta los siguientes comandos uno por uno:

sudo apt update && sudo apt upgrade
sudo apt install python3 python3-pip python3-venv

Verifica la instalación ejecutando:

python3 --version

Paso 2: Crear un Entorno Virtual

Un entorno virtual aísla las librerías de tu proyecto para evitar conflictos. Es una práctica esencial en el desarrollo con Python.

1. Crea la carpeta del proyecto y navega a ella

mkdir simulador_bolsa
cd simulador_bolsa

2. Crea el entorno virtual

python3 -m venv venv

3. Activa el entorno

El comando varía según tu sistema. Una vez activo, verás `(venv)` al inicio de la línea de comandos.

Windows (CMD): `.\venv\Scripts\activate`

macOS/Linux (Bash): `source venv/bin/activate`

Paso 3: Elegir tu IDE

Un Entorno de Desarrollo Integrado (IDE) te facilitará la escritura y depuración de tu código. VS Code y PyCharm son las opciones más populares.

Configurar Visual Studio Code (Recomendado)

VS Code es ligero, rápido y muy personalizable.

  1. Descarga e instala VS Code desde su sitio oficial.
  2. Abre VS Code, ve a la pestaña de Extensiones (icono de cuadrados).
  3. Busca e instala la extensión oficial "Python" de Microsoft.
  4. Abre la carpeta de tu proyecto (`simulador_bolsa`) en VS Code.
  5. VS Code debería detectar y seleccionar automáticamente tu entorno virtual `venv`.

Configurar PyCharm Community

PyCharm es un IDE potente, especializado en Python.

  1. Descarga e instala PyCharm Community Edition desde su sitio oficial.
  2. Inicia PyCharm y selecciona "New Project".
  3. En la ventana de creación, asegúrate de seleccionar "Previously configured interpreter".
  4. Haz clic en los tres puntos (...) y navega hasta el ejecutable de python dentro de tu entorno virtual: `simulador_bolsa/venv/bin/python` (o `Scripts\python.exe` en Windows).

Paso 4: Instalar Librerías

Con tu entorno virtual activado, instala las librerías necesarias. Solo necesitas elegir una opción por categoría.

A. Interfaz Gráfica (GUI)

Elige una para crear la ventana de tu aplicación.

Opción 1: PyQt5 (Escritorio Clásico)

pip install pyqt5

Opción 2: Kivy (Multiplataforma)

pip install kivy

B. Conexión a API

Para obtener datos de precios en tiempo real.

Opción 1: CoinGecko (Criptomonedas, sin clave API)

pip install pycoingecko requests

Opción 2: Alpha Vantage (Acciones y Cripto)

pip install alphavantage-api-client requests

C. Gráficos en Tiempo Real

Para visualizar los datos.

Opción 1: Matplotlib (Básicos)

pip install matplotlib

Opción 2: Plotly (Interactivos, recomendado)

pip install plotly kaleido

D. Base de Datos (SQLite)

Para guardar datos de usuarios y transacciones.

¡No necesitas instalar nada! El módulo `sqlite3` ya viene incluido en Python.

Paso 5: Prueba de Fuego

Verifica que todo se haya instalado correctamente. Crea un archivo `test_env.py` en la raíz de tu proyecto con este código y ejecútalo.

import sys

print(f"Versión de Python: {sys.version}")

try:
    # Sustituye 'PyQt5' por 'kivy' si elegiste Kivy
    import PyQt5
    print("✓ PyQt5/Kivy OK")
except ImportError:
    print("✗ Error en librería GUI")

try:
    import requests
    print("✓ requests OK")
except ImportError:
    print("✗ Error en requests")

try:
    import plotly
    print("✓ plotly OK")
except ImportError:
    print("✗ Error en plotly")

try:
    import sqlite3
    print("✓ sqlite3 OK")
except ImportError:
    print("✗ Error en sqlite3")

Ejecútalo desde tu terminal (con el entorno activado):

python test_env.py

Si ves `✓ OK` en todas las líneas, ¡estás listo para programar!

Paso 6: Organización del Proyecto

Una buena organización es clave. Aquí tienes dos pasos finales antes de empezar a codificar.

1. Genera `requirements.txt`

Este archivo guarda la lista de librerías de tu proyecto, facilitando su instalación en otras máquinas.

pip freeze > requirements.txt

2. Estructura de Carpetas

Una estructura recomendada para mantener tu código ordenado:

simulador_bolsa/
├── venv/
├── ui/
│   └── main_window.py
├── main.py
├── database.py
├── api_client.py
└── requirements.txt

¡Configuración completada! Mucho éxito en tu Proyecto Integrador.
